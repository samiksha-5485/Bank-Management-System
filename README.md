# Bank Management System

## 📌 Description
A console-based Bank Management System developed using Java and SQL to perform basic banking operations like account creation, deposit, withdrawal, and balance checking.

## 🚀 Features
- Create new bank account
- Deposit and withdraw money
- Check account balance
- View customer details
- Secure and structured data handling

## 🛠 Technologies Used
- Core Java (OOP Concepts)
- SQL (MySQL/PostgreSQL)
- JDBC (if used)

## 🧠 Concepts Used
- Object-Oriented Programming (Encapsulation, Inheritance)
- Exception Handling
- Database Connectivity
- CRUD Operations

## ▶️ How to Run
1. Open project in Eclipse / VS Code
2. Configure database connection
3. Run the main Java file
